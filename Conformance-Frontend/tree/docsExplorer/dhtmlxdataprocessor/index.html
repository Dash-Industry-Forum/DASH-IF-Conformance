DataProcessor

DataProcessor is a clear, functional library that lets you to ‘communicate’ with server-side backend. It not only simplifies work with server-side but also helps to monitor all data changes. You can use it with any DHTMLX component that allows editing inside itself. For example, grid or dataview.
Starting from dhtmlxSuite 2.1, there are ready-to-use PHP connectors which can be used instead of custom coding.
